# ProyectoJuegoODS
Juego basado en la concienzación de las ODS
1. ODS BENEFICIADOS:
  - HAMBRE 0
  - Ayuda sobre el Fin de la pobreza, al garantizar comida a la gente habría mayor cantidad de dinero para gastar en otras cosas y para tener más dinero.
2. ENTIDADES DEL CRUD
  - Usuarios
  - Puntuaciones
  - Formulario para contactar/sugerencias
3. DESCRIPCION:
  El juego se basa en tener comida en un lado de la pantalla la cual la irá cogiendo el personaje principal, y tendrá que hacersela llegar/lanzarla a la gente más necesitada, según sus gustos/pedidos si no simplemente entregarla, mientras esquiva obstáculos
